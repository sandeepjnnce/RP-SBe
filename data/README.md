\# Dataset Instructions



This research uses the following datasets:



\- \*\*rush\_hour\*\* and \*\*vidyo1\*\*: Download from \[Xiph.org Video Test Media](https://media.xiph.org/video/derf/)

\- \*\*PP4AV\*\*: Download from the official PP4AV repository



Place the videos in `data/videos/` folder.



\*\*Note\*\*: Due to size constraints, video files are not included in this repository.
